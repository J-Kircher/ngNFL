Help
This is a simple simulation of an NFL season. Statistics are tracked and playoff teams are determined from the outcome of the simulated season. After the playoffs a Super Bowl champion is crowned. Each team is ranked in nine categories:
Quarterbacks
Running Backs
Wide Receivers
Offensive Line
Defensive Line
Linebackers
Defensive Backs
Special Teams
Coaches
Using the schedule for the current season, the teams are compared against one another and "win" or "lose" based on the rankings and randomly generated numbers.
Here is a description of the navigation links:
Show Teams
This link will display all the NFL teams in the main section. Clicking on a team in this window will display that team's individual ratings and current statistics. The lower section will show the team's complete scheule.
Schedule
This link will bring up the league schedule. The active months are displayed in the upper section. Clicking on a date will display that day's matchups in the main section.
Standings
This link will display the current standings. The divisions are updated dynamically after each simulated game.
Standard/Expanded
Use these tabs to alternate the view of the standings.
Playoffs
This link is only displayed after the regular season. Clicking this link simulates the playoffs and displays the playoff bracket in the main section.
Bracket/Schedule
Use these tabs to alternate the view between the playoff bracket and the individual playoff games.
Top Teams
This link will display a popup showing the teams in the playoff hunt, along with teams on the bubble. This is also updated dynamically after each simulated game.
AFC/NFC
Use these tabs to alternate the view of the conference leaders.
Simulate
This button will begin simulating the games for the current day. The teams playing are shown in the top right section along with their scores and the quarter the game is in. All of the current day's games are displayed in the right section.
Sim Season
The first time you click the "Simulate" button, you are presented with a choice on how the simulation should proceed. There are two sliders, one to choose to simulate the entire season or one game at a time. The second will determine the speed of each simulated game. Clicking Cancel will run the simuation with the default options, one game at a time, and slower.
Settings
This button will open the "settings" dialog.
One Game at a Time
When on, you will only be allowed to simulate one game at a time. When off, you can simulate multiple games at once.
Sim Fast
When on, the games will simulate much faster.
Reset Season
This will allow you to return the simulation to its initial state, resetting the schedule and all games played.
Other helpful information:
Game Matchups
Clicking on any individual game will show either the "Matchup" or the "Results" dialog. If the game has not yet been played, then you will see the "Matchup" dialog. If the game is in progress, or has been completed, you will see the "Results" dialog.
Matchup
This dialog will show the head-to-head ratings for the two teams, as well as provide a point spread on the game.
Results
This dialog will show the results of the game, score by score through all four quarters.
